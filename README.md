# Namaste Green - Restaurant Website

🍃 **Global Healthy Cuisine - Asian & Middle Eastern Flavors**

A modern, responsive website for Namaste Green, an online-only delivery restaurant specializing in fresh salad bowls, trays, shakes, and smoothies with Asian and Middle Eastern flavors.

## 🌟 Features

- ✅ **Responsive Design** - Works perfectly on mobile, tablet, and desktop
- ✅ **Modern UI** - Beautiful gradient design with brand colors (#518b9c, #bdf3f3, #6ababb, #377b7c)
- ✅ **Smooth Animations** - Fade-in and scroll animations for better UX
- ✅ **Easy Navigation** - Sticky navbar with smooth scrolling
- ✅ **Menu Display** - Organized categories with prices
- ✅ **Order Buttons** - Direct links to Deliveroo and Uber Eats
- ✅ **Contact Info** - Central London location and opening hours (11am-3pm, 5pm-8pm)
- ✅ **Fast Loading** - Lightweight HTML, CSS, and vanilla JavaScript

## 📁 Files

- **index.html** - Main HTML structure
- **styles.css** - All styling with brand colors and animations
- **script.js** - Interactive features (smooth scrolling, scroll animations)
- **README.md** - This documentation

## 🚀 Deployment

### GitHub Pages (Free & Easy)

1. **Push files to your GitHub repo:**
   ```bash
   git add .
   git commit -m "Deploy Namaste Green website"
   git push origin main
   ```

2. **Enable GitHub Pages:**
   - Go to repository Settings
   - Scroll to "Pages" section
   - Select "Deploy from a branch"
   - Choose "main" branch
   - Click Save

3. **Your site will be live at:**
   ```
   https://dakshachodvadiya.github.io/bootstrap/
   ```

### Other Deployment Options

- **Netlify** - Connect GitHub repo, auto-deploy on push
- **Vercel** - Similar to Netlify, very fast
- **Custom Domain** - Update DNS to point to GitHub Pages

## 🔧 Customization

### Update Restaurant Links

Edit `index.html` and replace:
```html
<!-- Find these lines and update with your actual URLs -->
<a href="https://deliveroo.co.uk" target="_blank" class="btn btn-primary">Order on Deliveroo</a>
<a href="https://www.ubereats.com" target="_blank" class="btn btn-secondary">Order on Uber Eats</a>
```

### Change Menu Items

Edit the menu sections in `index.html`:
```html
<div class="menu-item">
    <h4>Your Dish Name</h4>
    <p>Your dish description</p>
    <span class="price">£9.99</span>
</div>
```

### Update Contact Information

Edit the contact section:
```html
<div class="contact-item">
    <h4>📍 Location</h4>
    <p>Your Address</p>
</div>
```
